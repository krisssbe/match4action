Our Vision

We are using 1,5 x earth’s resources every year. By 2050 this will be 3x if we continue living the way we do. 2 out of 5 kids still will never set foot in a classroom, the rich are getting richer, the poor poorer and there is continued high unemployment in one area where there are skills shortages in others.

If we approach these problems the way we have, they won’t get resolved. We need something new, we need disruption, we need to use technology, a fresh way of thinking, we need global players, who understand the local space. This is how Match4Action was born.
